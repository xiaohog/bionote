### 开发
```bash
yarn dev
```
### 提取新的翻译关键词
```bash
yarn extract
```

### 清空之前的所有翻译关键词对象并提取新的翻译关键词
```bash
yarn lang:extract
```

### 编译新的翻译关键词
```bash
yarn compile
```
